



# プロジェクトブリーフ：ascended336

## 概要
- **ブランド名：** ascended336
- **業種：** アパレル（新規ブランド）
- **目的：** ブランドの世界観訴求 + オンライン商品販売
- **ターゲット：** 10〜40代、ブリティッシュパンクファッション好き
- **ファッションイメージ：** 上品さの中に悪さ・荒々しさがある（Sex Pistols的）
- **納期：** 未定（練習案件・全力仕上げ）
- **予算：** なし
- **商品数：** 約30点
- **販売形態：** オンラインのみ

## サイト種別
ブランドサイト型EC（Next.js フロントエンド + BASE連携）
- 商品管理・決済は BASE を使用
- 購入時は BASE のチェックアウトへ遷移
- 公開後の商品追加・更新はクライアント（BASE管理画面）が担当

## ページ構成
- [x] `/` トップ（実装済み・データ連携済み／スタイリング仕上げは今後）
- [x] `/products` 商品一覧（BASE/モック連携済み・カテゴリーフィルタ動作）
- [x] `/products/[id]` 商品詳細（動的データ・動的SEO・BASE購入遷移）
- [x] `/lookbook` ルックブックギャラリー（ガワ実装済み・プレースホルダー画像）
- [x] `/story` ブランドストーリー（実装済み）

## 必須機能
- [x] BASE API 商品データ連携（`lib/base.ts`・トークン未設定時はモック30点へ自動フォールバック）
- [ ] Instagram投稿 埋め込み表示（未実装／フッターにリンクのみ）
- [~] SEO対策（メタタグ・OGP・動的generateMetadata済み ／ sitemap.ts・robots.ts・og:image 未）
- [x] スマートフォン対応（モバイルファースト・レスポンシブ済み）
- [~] スクロールアニメーション（Heroパララックス済み ／ 水平スクロールギャラリー 未）

## 対象外機能
- ニュースレター登録

## UIデザイン方針
- **テイスト：** ラグジュアリー × ブリティッシュパンク（上品さの中に反骨心）
- **ベースカラー：** `#0a0a0a`（ほぼ黒）
- **メインカラー：** `#f5f5f0`（オフホワイト）
- **アクセントカラー：** `#cc0000`（クリムゾンレッド）
- **フォント（見出し）：** Bebas Neue / Anton
- **フォント（日本語）：** Noto Sans JP
- **フォント（本文）：** Inter
- **アニメーション：** リッチ（Framer Motion）
  - 水平スクロールギャラリー
  - パララックス
  - 荒削りなテクスチャ演出

## 技術スタック
- **Framework：** Next.js 15（App Router）
- **Language：** TypeScript
- **Styling：** Tailwind CSS
- **Animation：** Framer Motion
- **Commerce：** BASE API + BASEチェックアウト
- **Instagram：** Instagram Basic Display API
- **Hosting：** Vercel

## 懸念事項・要確認事項
- BASE APIの利用にはBASEアカウントとアプリ申請が必要
- Instagram APIは個人 / ビジネスアカウントかで取得方法が異なる
- 多言語対応（日英）は今後の拡張として検討
